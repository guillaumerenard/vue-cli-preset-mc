# Preset MC

## How to use

### Create a new projet from github
```
vue create --preset guillaumerenard/vue-cli-preset-mc <app-name>
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
